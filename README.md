# 博客前端

基于 Vue3 + Typescript + Vite

## 安装

```shell
git clone https://github.com/zerowong/zerowong.github.io.git
yarn install
yarn dev
yarn build
yarn serve
```

> 如果你使用 VScode，需安装 Volar 扩展，然后开启`TS Plugin`并禁用 Vetur 扩展
